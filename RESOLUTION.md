# RESOLUTION — Protocol-Commercial

## Resolution record

### 2026-03-19 — v1.1.0 migration resolution

The repository was migrated from a v1.0.0-centric, shared-fragment layout to a v1.1.0 current line with flat, self-contained commercial schemas.

The resolution establishes that:

- `v1.1.0` is now the current normative release line
- `v1.0.0` remains retained only as a legacy published line
- commandlayer.org mirrors and Agent Card bindings should target the flat v1.1.0 request and receipt schema paths
- x402 references are now first-class commercial artifacts rather than side metadata

## Conflict rule

When docs, examples, and schemas disagree, the schema files and manifest control the normative interpretation, and the mismatch must be treated as release drift until corrected.
