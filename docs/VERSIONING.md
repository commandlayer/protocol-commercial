# Versioning

- SemVer on **schema families** (v1.0.0 for this release).
- Minor/patch may add optional fields; **never** breaking changes.
- Breaking changes => new major path `v2.0.0/…` (parallel).

**Stability gates**
- Tag `v1.0.0-candidate` → freeze after 2+ green CI cycles and 2+ external adopter confirmations.
