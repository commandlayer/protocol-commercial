# FAQ

**Q: Can I use Commercial for free?**  
A: Yes for evaluation and small projects below triggers. See EULA‑C2‑PRO.

**Q: Do I need receipts?**  
A: Yes. Every successful action must emit a verifiable receipt.

**Q: How do I publish to ENS?**  
A: See PUBLISHING.md; set ERC‑8004‑style TXT keys pointing to schema URLs.
